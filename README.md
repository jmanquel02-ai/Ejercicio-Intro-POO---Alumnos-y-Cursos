# Ejercicio Intro POO - Alumnos y Cursos

Este repositorio contiene el desarrollo del ejercicio introductorio de Programación Orientada a Objetos, donde se modelan alumnos de la UFRO y cursos que gestionan una lista de alumnos mediante ArrayList.

## Clases desarrolladas

- Alumno.java: representa a un alumno con sus atributos, constructores, getters, setters y método toString.
- Curso.java: representa un curso con nombre, código y una lista de alumnos. Permite agregar, eliminar y buscar alumnos por número de matrícula.
- Main.java: prueba la ejecución del programa creando un curso con al menos cinco alumnos.

## Diagrama UML

El diagrama de clases fue desarrollado usando PlantUML y se encuentra en el archivo `diagrama.puml`.

## Ejecución

Para ejecutar el proyecto, abrirlo en IntelliJ IDEA y ejecutar la clase `Main.java`.
